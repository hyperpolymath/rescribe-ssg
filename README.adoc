= rescribe-ssg
:toc: auto
:toclevels: 3

**The definitive ReScript static site generator.**

== Overview

rescribe-ssg is a type-safe static site generator written in ReScript.
It compiles to clean JavaScript while providing strong compile-time guarantees
through OCaml's type system.

== Language Enforcement

[IMPORTANT]
====
This SSG **MUST** be written in ReScript. No exceptions.

Forbidden languages for SSG logic:

* Python
* Plain JavaScript
* TypeScript (use ReScript)
* Any language other than ReScript

NOTE: This satellite is 100% ReScript - even the MCP adapter!
====

== Features

* Sound type system with inference
* Compiles to clean, readable JavaScript
* First-class JSX support for templates
* Pattern matching for content routing
* Excellent JavaScript interop

== Requirements

* Node.js 18+
* ReScript 11+

== Building

[source,bash]
----
npm install
npm run build
node lib/es6/src/RescribeSsg.mjs build
----
